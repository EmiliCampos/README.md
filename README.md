# CinePlus Emili Campos
CinePlus es un software de venta para un cine elaborado en el lenguaje Python con wxGlade y VS Code. Fue desarrollado por Emili Yoana Campos. El programa cuenta con 4 ventanas, las cuales son:
-	Inicio de sesión: Es la primera ventana del programa. Es la que permite ingresar al resto del programa si el usuario y la contraseña son correctos. 
-	Taquilla: Es donde se venden los boletos. Se puede seleccionar la película, el horario, la sala y la cantidad de boletos. Hay dos tipos de boletos: mayores y menores, cada uno con diferente precio.
-	Dulcería: Aquí se venden los combos y los productos individuales. Los combos son: básico, cuates, familiar. Las palomitas: chicas, medianas, grandes. Los refrescos, tamaño: chico, mediano, grande; los sabores: cola, Sprite, manzana. 
-	Ticket: Aquí se muestra la información de los productos y boletos vendidos de las ventanas anteriores. Es donde se procede a la parte del cobro. 
Para poder ejecutar el programa luego de descargar los archivos se debe hacer lo siguiente:
1.	Abrir VS Code
2.	Ir a “File”
3.	Dar click en “Select folder”
4.	Buscar los archivos y abrir esa carpeta
5.	Abrir “” y dar click en el botón de “play” en la parte superior derecha de VS Code
